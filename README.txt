English Learning Notes (Vue + Bootstrap 5)
==========================================

開啟 index.html 進入首頁；或直接進入 vocab.html / dialogue.html / reading.html。
在每個頁面左上角選擇日期即可切換不同天的內容。

部署方式：
- 可直接上傳整個資料夾至 GitHub Pages / Netlify / Vercel。
- 影片目前為標題展示，如需實際連結可再補上 URL。

資料檔：assets/data.js，若未來要追加新一天的內容，請依現有格式追加一筆物件。
